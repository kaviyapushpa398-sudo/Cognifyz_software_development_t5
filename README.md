# 📂 Task Manager with File Persistence (Python)

## 📌 Overview

This project is an enhanced version of the **Task Manager (CRUD Application)** built using Python. It includes **file handling (File I/O)** to store and retrieve tasks, ensuring data persistence even after the program is closed.

---

## 🎯 Objective

To implement persistent storage in a task management system using file handling, allowing users to save and load tasks from a file.

---

## 🏢 Project Source

This project was completed as part of a task provided by **Cognifyz**.

---

## 🛠️ Features Implemented

### ✅ 1. Persistent Storage

* Stores task data in a text file (`tasks.txt`)
* Saves data automatically after every operation

---

### ✅ 2. Load Existing Tasks

* Loads previously saved tasks when the program starts
* Ensures continuity of data

---

### ✅ 3. Task Management (CRUD)

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks

---

### ✅ 4. Structured Data Storage

* Stores task details in a structured format:

  * Task ID
  * Title
  * Description
  * Status (Pending / Completed)

---

### ✅ 5. File Handling

* Uses file read/write operations
* Automatically creates file if it doesn’t exist

---

### ✅ 6. Error Handling

* Handles file-related errors:

  * File not found
  * Read/write issues
* Displays user-friendly messages

---

### ✅ 7. Interactive Menu

* Menu-driven interface:

  1. Add Task
  2. View Tasks
  3. Update Task
  4. Delete Task
  5. Exit

---
